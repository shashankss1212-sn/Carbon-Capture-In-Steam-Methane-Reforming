# Life Cycle Assesment of Carbon-Capture-In-Steam-Methane-Reforming
This work presents a DWSIM simulation and OpenLCA life-cycle assessment of MEA-based CO₂ capture integrated with SMR-WGS hydrogen production. Capture efficiency, solvent loading, regeneration energy, and environmental impacts are evaluated to optimize low-carbon H₂ pathways.
